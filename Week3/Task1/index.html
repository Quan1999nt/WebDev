Câu 1: Những vị trí nào có thể đặt Javascript trong file HTML và dặt theo cú pháp nào? Nên đặt Javascript tại vị trí nào để có hiệu suất tốt nhất? Cho ví dụ minh họa.

  • Các thẻ <script> có thể xuất hiện trong phần <head> hay <body> của file HTML
  • Nếu thẻ <script> được đặt trong phần <head>, nó sẽ được tải và sẵn sàng trước khi phần còn lại của văn bản được tải

Trong Javascript có những kiểu dữ liệu nào? Làm thế nào để tạo một biến để lưu trữ giá trị tương ứng với những kiểu dữ liệu đó?

  • Tiêu chuẩn ECMAScript mới nhất xác định bảy kiểu dữ liệu:
    1. Kiểu boolean
    2. Kiểu của null
    3. Kiểu của undefined
    4. Kiểu số
    5. Kiểu chuỗi
    6. Kiểu Symbol
    7. Kiểu đối tượng

Câu 3: . Làm thế nào để truy xuất (lấy giá trị) trong 1 thẻ HTML bất kỳ và gán vào 1 biến để trong Javascript theo ID, theo Class, theo Name, theo tên thẻ HTML và theo CSS Selector (ví dụ .class, #id)? Cho ví dụ minh họa.

  • HTML Element by Id
    document.getElementById("demo").innerHTML = "Hello World";
  • HTML Elements by Tag Name
    var x = document.getElementsByTagName("p");
  • HTML Elements by Class Name
    var x = document.getElementsByClassName("intro");
  • HTML Elements by CSS Selectors
    var x = document.querySelectorAll("p.intro");

Câu 4: Xác định các lệnh Javascript tương ứng để thực hiện các yêu cầu sau::

  • Viết ghi chú trong Javascript (1 dòng, nhiều dòng)
    1 dòng dùng "//"
    nhiều dòng dùng "/*" và "*/"
  • Hiển thị thời gian hiện tại (từ năm đến giây theo múi giờ GMT+7 - Việt Nam).
    var d = new Date();
